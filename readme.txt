This is an encryptor/decryptor program coded in Ruby. Made using the tutorial on Jumpstart Lab: http://tutorials.jumpstartlab.com/projects/encryptor.html

It includes the following files:

encryptor_decryptor.rb - Real-time encryption/decryption from the command line. Includes a method for cracking encrypted text with an unknown cipher.

file_encryptor.rb - Encryption/decryption for saved files. Also includes a method for cracking encrypted text.

advanced_encryptor.rb - A more complex encryption that switches between three different ciphers.
